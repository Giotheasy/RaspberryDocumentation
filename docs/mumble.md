# Mumble Server

---

## Installation

To install and init:

```shell
sudo apt-get install mumble-server
sudo dpkg-reconfigure mumble-server
```

## Configuration

Finally, edit the server configuration file:

```shell
sudo vim /etc/mumble-server.ini
sudo /etc/init.d/mumble-server restart
```
